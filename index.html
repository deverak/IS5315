What is Docker

Docker is an open-source project that allows developers and system administrators to quickly access and deploy new versions of web applications under different development enviornments, virtually anywhere. For example, with Docker, an individual would be able to build an application on a Windows laptop inside a Docker container and then take an image of that container and deploy it to a Linux production enviornment. This reduces the dependency for the main system, meaning that a person would not have to consider whether or not they will be installed as the same operating system the application was built on.

Docker is a computer program that performs "containerization." A container does not virtualize the hardware but instead packages up the code and its dependencies to ensure efficiency and reliability from one operating system to another.

Source: https://www.docker.com/resources/what-container; https://docs.microsoft.com/en-us/dotnet/standard/microservices-architecture/container-docker-introduction/docker-defined; https://www.zdnet.com/article/what-is-docker-and-why-is-it-so-darn-popular/

What is Nginx

Originally Nginx was designed as a web server to optimize performance and stability. Now, Nginx is an open-source web server and a multifunctional tool that can be deployed as a reverse proxy, HTTP cache, and load balancer. Nginx is able to better manage incoming traffic and distribute connections to slower upstream servers because of its abiity to manage a high volume of connections.

Source: https://www.nginx.com/resources/glossary/nginx/

What is VM (virtual machine)

A Virtual Machine or "image" stimulates a computer system. Virtual machines provide the same functionality of a physical cpmputer and gives the end user the same experience as a host operating system itself. This is an optimal enviornment for testing other operating systems such as beta releases and running applications on operating systems that differ from the intended operating system.

Source: https://azure.microsoft.com/en-us/overview/what-is-a-virtual-machine/

Who is Cloud Computing Service Provider - Digitalocean

DigitalOcean is a cloud computing service that provides an Infrastructure as a Service. This means that DigitalOcean simplifies managing infrastructure for developers and teams while offering the option to deploy multiple virtual machines or "Droplets." DigitalOcean is a reliable platform that allows developers to scale, rebuild, and redirect network traffic between droplets.

Source: https://www.digitalocean.com/
